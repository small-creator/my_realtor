#### 3.1 전문직 비교 섹션
3개 카드 레이아웃으로 구성:
- **변호사** (✅ 전문가 먼저 선택) - 녹색 테두리
- **의사** (✅ 전문가 먼저 선택) - 녹색 테두리
- **부동산** (❌ 매물에 딸려옴) - 빨간색 테두리

각 카드는 4단계 프로세스 포함

#### 3.2 글로벌 스탠다드 섹션
파란색 하이라이트 박스로 구성:

**미국 vs 한국 프로세스 비교**
- 🇺🇸 미국: Buyer's Agent 시스템
  1. 여러 중개인 인터뷰 후 선택
  2. Buyer's Agent와 계약 체결
  3. 에이전트가 시장 전체 검색
  4. 매물 추천 및 협상 대리

- 🇰🇷 한국 현실
  1. 직방에서 매물 검색
  2. 매물에 딸려온 중개사 만남
  3. 그 매물만 보게 됨
  4. 누군지도 모르는 중개사에 의존

**미국 시스템의 장점**
- 이해상충 방지: 중개인은 매수인만을 위해 일함
- 시장 전체 접근: MLS를 통해 모든 매물 비교 가능
- 전문성 검증: 경력과 평판을 직접 검토
- 비용 효율: 중개 수수료는 판매자 부담(5~6%)

**마무리 메시지**
- "한국도 이제 바뀌어야 합니다"
- "마이중개사가 그 시작입니다"
