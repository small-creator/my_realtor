# "매물을 찾으면 그 매물만 팔려는 사람을 만납니다" 카피 빌드업

## 🎯 선택된 카피의 강점

**"매물을 찾으면 그 매물만 팔려는 사람을 만납니다"**

**왜 강력한가?**
- ✅ 문제의 본질을 한 문장으로 정확히 지적
- ✅ 고객이 경험했을 불편함을 언어화
- ✅ "아, 그래서 그랬구나" 깨달음 제공
- ✅ 자연스럽게 해결책(마이중개사) 유도
- ✅ 짧고 강렬하며 기억하기 쉬움

---

## 🏗️ 빌드업 버전 (단계별)

### 버전 1: 기본 확장형

**메인 카피:**
```
매물을 찾으면
그 매물만 팔려는 사람을 만납니다.
```

**서브 카피:**
```
전문가를 찾으면
모든 매물을 보여주는 사람을 만납니다.
```

**CTA:**
```
이제 전문가를 선택하세요.
```

---

### 버전 2: 문제 심화형 ⭐

**메인 카피:**
```
매물을 찾으면
그 매물만 팔려는 사람을 만납니다.
```

**빌드업:**
```
더 좋은 집이 있어도
알려주지 않습니다.

단점이 있어도
숨기고 싶어합니다.

그게 자기 매물이니까요.
```

**전환:**
```
이제 바꾸세요.
```

**해결책:**
```
전문가를 먼저 선택하면
시장 전체를 볼 수 있습니다.

마이중개사는 팔 매물이 없습니다.
오직 당신에게 맞는 집만 찾습니다.
```

---

### 버전 3: 대비 강조형 ⭐⭐

**메인 카피:**
```
매물을 찾으면
그 매물만 팔려는 사람을 만납니다.
```

**대비:**
```
❌ 직방에서 매물을 찾으면
   → 그 매물을 올린 중개사
   → 그 매물만 보여줌
   → 다른 선택지 없음

✅ 마이중개사를 먼저 선택하면
   → 시장 전체를 아는 전문가
   → 모든 매물을 비교
   → 최선의 선택 가능
```

**결론:**
```
순서를 바꾸세요.
매물이 아니라 전문가를 먼저 선택하세요.
```

**전환점:**
```
이제 바꿔야 합니다.
```

**해결책:**

```
대신 시장 전체를 찾아드립니다.

모든 매물을 비교하고
장단점을 솔직하게 알려드리며
당신에게 맞는 집을 찾아드립니다.

이게 부동산을 하는 올바른 방법입니다.
```

**CTA:**

### 버전 5: 질문 연속형 ⭐⭐

**진실:**

```

```


### 버전 6: 통계/사실 강조형 ⭐

**메인 카피:**
```
매물을 찾으면
그 매물만 팔려는 사람을 만납니다.
```

**팩트:**
```

그 매물을 올린 중개사에게 연락합니다.

그 중개사는 그 매물을 팔아야 합니다.
더 좋은 매물이 있어도
다른 선택지를 제시할 이유가 없습니다.
```

**비교:**
```
미국은 다릅니다.
매수자의 88%가 자신의 에이전트를 먼저 선택합니다.

그 에이전트는 매물이 없습니다.
시장 전체를 탐색해서
매수자에게 최적의 집을 찾아줍니다.
```

**전환:**
```
한국도 이제 바뀌어야 합니다.

마이중개사가 그 시작입니다.
```
